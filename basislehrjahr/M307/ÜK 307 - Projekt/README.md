## AODrive-Projekt von Aron Schneider & Olivier Winkler

Beschreibung
-------------
- Webprojekt für einen Cloud-Dienst. 
- Auf unserer Cloud kann ein Benutzer seine persönlichen Dateien wie Bilder, Videos etc. hochladen und verwalten.
- Auf unserer Webseite kann man einen Account erstellen.
